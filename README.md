# RentSafe Scanner (Gemini Flash 2.0) — Chrome Extension

Paste (or upload) a rental agreement and get:
- A **safety score (0–100)**
- **Top red flags**
- **Clause-by-clause** plain‑English explanations, missing tenant protections, excessive penalties, and possibly illegal terms

## Setup

1. Open Chrome → `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder: `text-highlighter-saver`

## Add your Gemini API key

1. Click the extension → **Options**
2. Paste your **Google AI Studio** API key
3. Click **Save**

## Demo flow (for your YouTube video)

1. Open the extension popup
2. Paste a rental agreement (or upload a `.txt`)
3. Click **Analyze**
4. Show:
   - Safety score
   - Top red flags
   - Clause-by-clause output (with High/Medium/Low)

## Notes / constraints

- This MVP expects **plain text** input (copy/paste or `.txt` upload).
- Output is informational; laws vary by jurisdiction.
